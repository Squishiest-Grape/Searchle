How to Use
The main box is the pattern box
Use '?' or '#' or '*' or '_' or '.' as wildcard
Use #, #-#, #+ in front to specify the allowable count
Use the requires, avoids, and allows to limit wildcards
Use '[]' or '()' to specify in-place wildcard limits
Use '!' or '~' or '^' to disallow values
Use '' or "" to make substring values
Spaces, commas, and capitalization is ignored
